Как запустить проект

1. Перейти в папку backend
2. В консоли выполнить команду npm i && npm run start:dev
3. Перейти в папку frontend
4. Открыть index.html
4. Для открытия ЛК сотрудника Марии(для сценария ниже) дописать в адресной строке ?userId=4

Архитектура проекта
1. Blockhain API, предоставленный ВТБ
2. Gamification BE, лежит в папке backend
3. Gamification UI, лежит в папке frontend
4. База данных Postgres, развернутая в облаке Amazon
